Alien Invasion

  • Source code

Alien Invasion is a simple arcade style game written in Python using pygame library. It has a scoring system and multiple levels of difficulty. It was inspired by Eric Matthes's "Python Crash Course".

Alien Invasion can be played locally after a simple set up process that can be found along with the controls in the project's README on GitHub.
