Learning Log

  • Source code

Learning Log is a simple Django application written in Python for creating a learning progress log. It has a registration and log in functionality with an admin page for the superuser. Users can add/edit Topics and Entries in the respective topics.

Learning Log app can be deployed locally following step by step instructions in the README.
